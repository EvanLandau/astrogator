This file is the specification for the object format to be used.
Object definitions are taken from the files in this folder, except for the central body, which is included in the file "center", which has no file type. On line one the mass is listed. On line two the radius is listed.

File names:
earth.json
jupiter.json
saturn.json
uranus.json
neptune.json
mars.json
venus.json
mercury.json
belt.json
others.json

Each file can contain data for multiple bodies.

Files are written in json format, with the schema defined in object_format.schema.json
