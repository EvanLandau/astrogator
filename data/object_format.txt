This file is the specification for the object format to be used.
Object definitions are taken from the files in this folder

File names:
center.json
earth.json
jupiter.json
saturn.json
uranus.json
neptune.json
mars.json
venus.json
mercury.json
belt.json
others.json

Each file can contain data for multiple bodies, except for the center file, which always holds data for the central body of the system

Files are written in json format, with the schema defined in object_format.schema.json
